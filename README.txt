Телеграм чат-бот «Tecno» предназначен для сбора текстовых сообщений (пожеланий) в определённом университете. 
Пользователь сможет перейти по qr-коду с заранее заданными параметрами (метками) своего университета, 
оставить текстовое сообщение (пожелание), которое попадает в google -таблицу с указанным ниже набором полей.
Необходимо создать для каждого университета свой QR-код с параметрами (31 QR-code), 
передаваемыми боту для определения информации об университете. 
Данные университета записываются в google-таблицу вместе с текстовым сообщением (пожеланием) и 
именем пользователя. Каждый параметр (текстовое сообщение, имя, вуз) располагается в отдельной ячейке google-таблицы.
Полученные с помощью чат-бота данные модерируются вручную. После проверки введенных данных, 
модератору необходимо ввести код для чат-бота, 
для отправки текста об успешной или неуспешной отправке, а также отправлять открытку в формате gif.
Реализовать выбор анонимности публикации. Пользователь может анонимно отправить текстовое сообщение во все университеты, 
либо не анонимно в свой университет. 
Чат-бот состоит из 12 сообщений, которые появляются в зависимости от выбора пользователем. 
Пользователь вводит следующие данные:
4.1. Имя
4.2. Пожелание
4.3. Указывает анонимность пожелания

При любом исходе событий необходимо довести до пользователя ссылку на ресурс «TECNO» 
 В случае если пользователь запустил бота и отказался от написания пожелания, 
боту необходимо прислать через неделю уведомление - напоминание пользователю о написании пожелания.
После отправки пользователем пожелания и перед записью в google-таблицу, 
чат-бот должен отфильтровать сообщения, содержащие нецензурную лексику (на основе имеющихся словарей нецензурной лексики).
